Face Recognition System using Python

Project Overview

This project is a Face Recognition System developed using Python and ANN Model training.
It detects human faces from images and their labels and recognizes individuals by comparing them with a trained dataset.
The system automatically identifies known faces from a stored dataset and displays recognition results in real time.


Objective:-
To build an intelligent system capable of:
Detecting human faces
Training on face datasets
Recognizing individuals accurately
Demonstrating practical application of Machine Learning & AI.

Features:-
 Face Detection using OpenCV
 Face Recognition from Dataset
 Image Processing & Feature Extraction
 Real-time Recognition Output
 Easy Dataset Integration
 Easy to use output interface


Technologies Used:-
Python
OpenCV
NumPy
Scikit-learn
Matplotlib

Installation & Setup:-
1. Clone Repository

2️. Navigate to code file 
main.py

3️. Install Dependencies
pip install -r requirements.txt

4️. Run Project
python main.py



Dataset:-
The dataset contains labeled facial images used for training the recognition model.
You can add new faces inside:
dataset/dataset/faces/
Each folder represents one person.




Output
The system:
Detects names/images' path from given data by the user
Matches faces with trained dataset
Displays recognized person's selected type of image/image path
